The program is compiled and run using the mingw32-make command

When the program is first opened
If you press (1), it creates everything from scratch randomly
If you press (2), it opens the last saved file and continues from where you left off

*LIST MODE
W and S Keys: Allows you to navigate up and down between nodes in the doubly linked list
C Key: Completely deletes the selected box (node)
F Key: Enters the selected box and allows you to manage the shapes on the right (Switches to Shape Mode)
ESC Key: Saves everything you have done and exits the program

*SHAPE MODE
W, A, S, D Keys: Moves the selected shape up, left, down, and right
Q and E Keys: Switches between other shapes inside that box
C Key: Deletes the selected shape. (If you delete the last shape, the box is also deleted)
G Key: Allows you to go back, switches to List Mode again
